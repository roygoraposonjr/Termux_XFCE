Usage

After installation, use the following commands to manage your XFCE desktop environment:

Start XFCE

start

Starts Termux native XFCE Desktop

Enter Debian Proot

debian

Logs into the Debian proot-distro

Run Debian Apps

prun <application>

Example: prun /opt/vesktop/vesktop --no-sandbox

Run apps with zink

zrun <application>

Example: zrun /opt/minecraft/AppRun
